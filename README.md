# AetherScore - 智能科技霓虹計分牌

一個專為手機、平板等行動裝置設計的現代化、極簡霓虹風格 RWD 網頁計分板。採用原生 HTML5、CSS3 與 JavaScript 開發，具備豐富的微動畫、手勢操作與流暢的音效反饋，非常適合各類運動比賽、桌遊或娛樂計分使用。

## 🌟 核心特色

### 📱 完美行動裝置適配 (RWD)
- 支援手機與平板橫置（Landscape）及直置（Portrait）擺放。
- 直置時局數控制區自動收縮至右側，最大化分數顯示面積；橫置時兩側分立，完美契合賽場視角。

### ⚡ 直覺的手勢操作 (Gestures)
- **分數板塊**：輕點即可加分，向左滑動（或滑鼠向左拖拽）即可減分。
- **局數板塊**：
  - 向上滑動（或向上拖拽）觸發加局，會彈出確認視窗詢問是否「開啟新一局並歸零分數」（舊局分數會被封存）。
  - 向下滑動（或向下拖拽）觸發回退，會自動還原上一局結束時的分數，便於修正。
- **名稱編輯**：雙擊（或行動端雙點擊）隊伍名稱，即可直接進行雙擊編輯（支援最多 6 個字元/約 3 個中文字）。

### 📊 實時各局分數紀錄
- 中間面板底部設有「各局分數」按鈕，點擊彈出玻璃擬態的表格。
- 表格展示各局分數（包含當前實時局數）與總得分。
- **字型自適應**：列數增多時自動等比例縮小字型，保證小螢幕下資料完全不折行。

### 🔊 科技感合成音效
- 基於 Web Audio API 即時合成的電子音效（非載入音訊檔案，速度極快、零延遲）。
- 向上加分、向下減分、開新局、系統操作均有專屬音效，支援一鍵靜音。

### 🔄 進階功能
- **對調按鈕**：點擊可一鍵將兩隊的名稱與分數水平鏡像對調，並伴隨酷炫的翻轉動畫。
- **撤銷系統**：支援多達 50 步的歷史操作撤銷（Undo），隨時回滾任何誤觸。
- **狀態持久化**：自動儲存資料至瀏覽器本地快取（LocalStorage），意外重新整理網頁也能無縫繼續。

## 🚀 快速開始

### 本地執行
本專案為純前端應用，無需安裝任何依賴。您只需將專案複製到本地，雙擊 `index.html` 即可直接在瀏覽器中執行。

若要體驗完整功能（如 LocalStorage 和 Web Audio 在部分瀏覽器的安全限制），建議使用輕量本地伺服器啟動：
```bash
# 使用 Python 啟動本地伺服器
python -m http.server 8000
```
然後在瀏覽器中開啟 [http://localhost:8000](http://localhost:8000)。

## 🌐 部署至 GitHub Pages

由於專案為靜態網頁，可以免費部署至 GitHub Pages：

1. 在 GitHub 上建立一個新的公開倉庫（Repository）。
2. 將本專案程式碼推送到該倉庫的 `main` 分支。
3. 進入倉庫的 **Settings** -> **Pages**。
4. 在 **Build and deployment** 下，將 Source 設定為 **Deploy from a branch**，分支選擇 **`main`**，目錄選擇 **`/ (root)`**。
5. 點擊 **Save**，稍等 1-2 分鐘即可透過 GitHub 提供的網址線上訪問計分板！
